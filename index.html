Sublime Text Completions for Chai Library

This Sublime package will install completions for bdd and tdd:

  • should
  • expect
  • assert

You can read the full API for chai syntax.

The completions work in both Javascript and CoffeeScript.

This package was forked from seethroughtrees's package, sublime-chai-full-completions.

Installation

cd ~/Library/Application\ Support/Sublime\ Text\ 3/Packages/
git clone git://github.com/sbolel/sublime-chai-completions.git


Improvements and suggestions are welcome!
